# Minicode (中文版)

Minicode 是一个顶尖的 AI 编程助手，旨在通过命令行界面自主协助完成软件工程任务。

## 特性

- **自主任务执行**：能够独立阅读、编写和编辑代码。
- **工具集成**：配备了用于文件操作、代码搜索和系统命令执行的专业工具。
- **自我意识**：能够分析并修改自身的代码库。
- **专业客观性**：专注于技术准确性和直接解决问题。

## 项目结构

- `main.py`: 应用程序入口，处理 CLI 循环。
- `agent.py`: 包含 `MinicodeAgent` 类，管理对话流和工具分发。
- `provider.py`: 与 LLM 交互（当前配置为 OpenRouter）。
- `prompts.py`: 定义系统指令和行为准则。
- `tools/`: 原子工具集合：
  - `read.py`: 读取文件内容的工具。
  - `write.py`: 创建新文件的工具。
  - `edit.py`: 在现有文件中进行精确搜索替换编辑的工具。
  - `search.py`: 在目录中搜索代码的工具。
  - `bash.py`: 执行系统命令的工具。

## 入门指南

### 前置条件

- Python 3.12+
- OpenRouter API 密钥（或兼容 OpenAI 协议的提供商）。

### 安装

1. 克隆仓库。
2. 在根目录下创建 `.env` 文件并添加你的 API 密钥：
   ```env
   OPENROUTER_API_KEY=your_api_key_here
   ```
3. 安装依赖（如有）。

### 使用方法

使用以下命令运行助手：
```bash
python main.py
```
